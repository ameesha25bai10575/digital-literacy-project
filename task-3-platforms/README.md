Task 3: Coding and Collaboration Platforms

Platforms Used:
For this task, I explored HackerEarth for coding practice and Google Workspace tools (Google Forms and Google Sheets) for digital collaboration. These platforms are very useful for computer science students because they help improve programming skills as well as data management and teamwork. HackerEarth provides coding challenges that strengthen logical thinking, while Google Workspace tools help in creating forms, collecting responses, and organizing data efficiently.

Project Overview:
In Part A, I used HackerEarth to solve a programming problem where I had to find the smallest prime number that divides two given integers. The task required analyzing both numbers and determining whether they share a common prime factor. If a common prime divisor exists, the program prints the smallest one; otherwise, it prints -1. For example, when the input values were 3 and 6, the output was 3, since 3 is the smallest prime number that divides both numbers. Solving this problem helped me practice logical thinking and better understand prime numbers and problem-solving using code.

In Part B, I created a Digital Literacy Awareness Quiz using Google Forms. The quiz included five questions with different formats such as multiple-choice and short-answer questions. I then linked the form responses to Google Sheets, which automatically collects and organizes the responses in a spreadsheet for easy analysis.
